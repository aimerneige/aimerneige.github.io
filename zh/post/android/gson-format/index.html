使用 GsonFormat 插件快速构建 Json 映射对象

2020年2月16日 · Aimer Neige |  Suggest Changes
目录
  • 插件安装
  • 使用方法

插件安装#

  1. File
  2. Settings
  3. Plugins
  4. Marketplace
  5. 在搜索框输入 GsonFormat 搜索
  6. 安装相应插件
  7. 根据提示重启 IDE

使用方法#

  1. 新建实体类
  2. ALT + Insert 或者 右键->Generate
  3. 选择 GsonFormat
  4. 将待解析 Json 字符串输入弹出的对话框
  5. OK
