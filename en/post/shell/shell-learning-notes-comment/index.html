Shell Comment#

Start with #

# this is a line of comment

Simply write more with # start

# Author: Aimer Neige
# Email: aimer.neige@aimerneige.com
# Date: 2022-04-29

Other way:

:<<EOF
Author: Aimer Neige
Email: aimer.neige@aimerneige.com
Date: 2022-04-29
EOF

:<<'
Author: Aimer Neige
Email: aimer.neige@aimerneige.com
Date: 2022-04-29
'

:<<!
Author: Aimer Neige
Email: aimer.neige@aimerneige.com
Date: 2022-04-29
