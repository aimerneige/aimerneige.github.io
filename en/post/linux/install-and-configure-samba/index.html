SMB File Share#

Update System#

sudo apt update

Install Samba#

sudo apt install samba

Edit Share Config#

sudo vim /etc/samba/smb.conf

Add share point at the end of the config file

[share-name]
    comment = Your Comment Here
    path = /path/to/share/dir
    read only = no
    browsable = yes

[public]
    comment = Public Shared Disk
    path = /mnt/disk/smb/share
    read only = no
    browsable = yes
    create mask = 0644
    directory mask = 0755

[private]
    comment = Pricate Data Share
    path = /mnt/disk/smb/private
    read only = no
    browsable = no
    valid user = aimer, umb
    create mask = 0644
    directory mask = 0755

[media]
    comment = Media Files
    path = /media
    read only = yes
    browsable = yes

Start Samba#

sudo systemctl start smbd.service

Enable Autostart#

sudo systemctl enable smbd.service

Update Firewall Rules#

sudo ufw allow samba

Setting User Password#

sudo smbpasswd -a username

Username used must belong to a system account, else it won‚Äôt save.

Reset User Password#

sudo smbpasswd -a user-name
